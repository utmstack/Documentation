# Integration guide for MacOS

*The macOS agent collects system and application logs, monitors activity, and communicates securely with the UTMStack master server or proxy. It also enables incident response actions.*

> **Note:** This feature is part of a **paid subscription**. Please contact our support team to obtain the required installation dependencies.


## Installation Steps


### 1. Configure your device to send logs to a UTMStack agent on ports:
        * 9000 
        * 9001
        * 50051

### 2. Contact Support.


> **Note:** Reach out to UTMStack support to request the **macOS installation dependencies**. These files are necessary to perform installation or uninstallation of the agent.
